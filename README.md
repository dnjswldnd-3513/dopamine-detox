# dopamine-detox

현재는 간단한 계획 느낌으로 정리

목표는 숏폼 컨텐츠(유튜브 숏츠, 틱톡, 릴스) 사용시간을 측정하고 설정한 목표 시간을 초과하면 경고를 띄워즌 디지털 디톡스 앱 백엔드


---

## 핵심 기능
1. 숏폼 사용 시간 추적
    - 앱 사용 시작/ 종료 기록
    - 자동으로 사용 시간 계산
2. 목표 설정
    - 앱 카테고리별 하루 제한시간 설정
    - 목표 초과 시 자동 감지
3. 알림 시스템
    - 목표 초과 시 fcm 푸시 알림 발송
    - 경고 내역 조회
4. 챌린지 시스템
    - Admin이 기간제 챌린지 생성
    - 유저가 참여 후 달성시 보너스 점수 획득
5. 점수 & 랭킹
    - 사용 시간 적을수록 점수 획득
    - 챌린지 달성 시 보너스 점수
    - 점수 기반 주간 랭킹
6. 회원 관리
    - 회원가입 / 로그인 (jwt 인증)
    - Admin / user 권한 분리

## 기술 스택
- JAVA 17
- Spring Boot 3.5.x
- Spring Security+JWT
- Spring Data JPA
- MySQL
- Lombok
- Swaager
